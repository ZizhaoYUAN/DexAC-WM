Abstract

Recent advances in action-conditioned world models show promising progress in modeling complex interactions and forecasting future states under diverse action sequences. While these models are often driven by stronger visual representations and model capacity, action conditioning itself remains underexplored.

Most existing approaches compress entire action sequences into a single representation, which works well for low-DoF control but becomes less reliable in high-DoF scenarios. We observe that high-DoF dexterous actions are inherently heterogeneous, spanning multiple orders of magnitude, where large-scale motions coexist with subtle but important signals. When uniformly aggregated, optimization exhibits an imbalance across action components, which hinders the modeling of fine-grained effects and affects action fidelity.

We therefore propose DexAC, which treats action conditioning as a structured process rather than global compression. Our method preserves dimension-level semantics via action tokenization and aligns action signals with visual dynamics through an adaptive framework combining local refinement and global modulation. Experiments on EgoDex show consistent improvements in visual fidelity and action consistency, suggesting that scaling world models to high-DoF control depends not only on model capacity but also on how heterogeneous actions are structured and integrated.

The Problem

Why vanilla action conditioning breaks at high DoF

Low-DoF gripper actions (6 dimensions) are neatly bounded — all dimensions live at the same scale, so global MLP aggregation works fine. Dexterous human hands are a different story.

[ Fig. 2a: Low-DoF (6-DoF) action distribution ]
Low-DoF (6-DoF). Translation and rotation dimensions maintain bounded variances at ~10⁻² scale — gradients stay balanced.
[ Fig. 2b: High-DoF (57-DoF) action distribution ]
High-DoF (57-DoF). A 10⁵ scale gap exists between macro-movements (wrist/camera) and micro-movements (fingers), creating gradient dominance and suppressing subtle signals.

When all 57 action dimensions are flattened into a single embedding, high-variance wrist/camera dimensions dominate the gradient signal and effectively silence the finger articulations — the very signals needed for dexterous manipulation. This heterogeneous semantic collapse manifests as slower convergence and higher training loss.

Method

DexAC: Structured Action Conditioning

DexAC replaces global compression with a structured pipeline that keeps each action dimension semantically independent throughout the conditioning process.

DexAC architecture. Actions are independently normalized per dimension and tokenized into a structured set of (B, N, C) tokens. Two conditioning branches then inject action information into each DiT block: a local cross-attention branch for fine-grained per-dimension alignment, and a global AdaLN branch for temporally coherent motion intent. A semantic condition branch fuses DINOv3-L dense features with VLM text embeddings via dual cross-attention, providing strong spatial-semantic grounding.

Structured Action Representation

Each action dimension is independently normalized and temporally tokenized, preserving per-dimension semantics and eliminating scale-induced gradient dominance across 57-DoF actions.

Unified Local-Global Conditioning

Local cross-attention injects fine-grained tokens directly into noise tokens; global AdaLN summarizes overall action intent. Both branches together capture dexterous detail and long-range coherence.

Semantic Condition Branch

DINOv3-L dense spatial features and VLM text embeddings are fused via dual cross-attention, providing complementary fine-grained spatial grounding and high-level semantic intent to stabilize generation.
